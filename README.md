# EasyVer

EasyVer 是一款完全本地化的、支持任意文件类型的现代图形化版本管理工具。

不同于 Git 这样面向代码文本的复杂工具，EasyVer 致力于为设计师、策划、文档编写者以及独立开发者提供一个**直观、简单、开箱即用**的文件版本管理体验。它可以轻松处理各种二进制文件（如图片、视频、设计源文件、压缩包等）的版本快照。

---

## 核心特性

- **现代化的 Fluent UI 设计**：基于 `PyQt6-Fluent-Widgets`，提供流畅、美观的暗色/浅色双主题无缝切换。
- **全文件类型支持**：底层基于强大的对象存储与哈希去重机制（利用 `zstandard` 压缩），无论文本还是二进制大文件，均能高效管理。
- **直观的时间线视图**：清晰的“里程碑（Milestone）”卡片流展示版本历史，支持名称/说明搜索以及日期范围筛选。
- **多彩标签系统**：为重要的版本打上自定义名称与颜色的 Tag，支持通过 Tag 快速筛选版本。
- **文件树实时监控**：基于 `QFileSystemWatcher` 的工作区目录监听，实时感知文件的增、删、改状态。
- **选择性版本暂存**：创建新版本时，通过打勾轻松挑选需要记录进里程碑的文件。
- **灵活的还原与提取**：支持一键将整个工作区恢复至某次里程碑，或从历史版本中单独提取某个被覆盖的文件。
- **多仓库并行管理**：支持记录并快速打开最近使用的多个本地仓库，互不干扰。

---

## 技术栈

- **前端框架**: [PyQt6](https://www.riverbankcomputing.com/software/pyqt/) + [PyQt6-Fluent-Widgets](https://github.com/zhiyiYo/PyQt-Fluent-Widgets)
- **底层存储**: 本地文件系统 + SQLite 关系数据库 (记录版本元数据)
- **核心算法**: 文件哈希对比去重 (Hasher) + `zstandard` 极速压缩算法

## 安装与运行

### 1. 环境准备
请确保您的计算机上已安装 **Python 3.11** 或更高版本。

### 2. 克隆项目
```bash
git clone https://github.com/your-username/easyver.git
cd easyver
```

### 3. 安装依赖
本项目使用 `poetry` 进行依赖管理，但也提供了标准的 `requirements.txt`。

**方式 A：使用 pip (推荐开发测试使用)**
```bash
pip install -r requirements.txt
```

**方式 B：使用 poetry**
```bash
poetry install
```

### 4. 编译资源文件 (如果修改了 QSS 或图片)
```bash
# 请将 pyside6-rcc/pyrcc6 替换为您环境下的实际路径
pyside6-rcc resources/resource.qrc -o app/resource_rc.py
```

### 5. 启动应用
```bash
python main.py
```

## 核心功能概览

### 创建仓库
在启动页点击“新建仓库”，选择本地任意一个空文件夹（或已有文件的文件夹）作为工作区，输入仓库名称即可初始化。系统会在该目录下生成一个隐藏的 `.easyver` 文件夹用于存放数据库和压缩对象。

### 记录里程碑 (Commit)
进入仓库页后，左侧文件树会自动扫描出变更的文件。点击右上角的“创建里程碑”，在弹窗中勾选需要保存的文件，输入版本说明，点击创建即可。

### 管理标签 (Tag)
在里程碑详情面板中，点击“打标签”可为当前版本添加颜色醒目的标签。在中间的卡片列表中，点击顶部的“标签筛选”按钮即可按标签过滤版本。

### 还原与提取 (Restore / Extract)
- **整体还原**：在详情面板点击“整体还原”，将会把当前工作区的所有文件回滚到该里程碑的状态。
- **提取文件**：如果只想找回某个被删改的历史文件，点击“提取文件”，选择文件并另存为即可，不会破坏当前工作区。

## 目录结构

```text
easyVer/
├── app/                  # 应用生命周期、全局配置与单例管理
├── core/                 # 核心业务逻辑：快照、差异对比、哈希去重、存储
├── db/                   # 数据库连接、迁移脚本及 DAO 层
├── resources/            # QSS 样式表、图标等静态资源
├── ui/                   # GUI 视图层
│   ├── dialogs/          # 弹窗与对话框
│   ├── pages/            # 路由主页面 (欢迎页、仓库页、设置页)
│   └── widgets/          # 局部可复用组件 (文件树、卡片、时间线)
├── utils/                # 通用工具函数 (时间处理、文件 IO、图标映射)
└── main.py               # 应用程序入口
```
